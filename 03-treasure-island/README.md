# Day 3: Treasure Island

## Assignment
> Create a Choose Your Own Adventure game. The program leads the user through a series of logical decisions where each choice determines if they continue their quest for treasure or face an early Game Over.

## Requirements
> The program must use nested conditional statements to create multiple story branches.
> Convert user input to lowercase using the .lower method to handle case sensitivity.
> Implement logic for winning and several losing scenarios based on specific choices.
> Display a welcome message using ASCII art to enhance the game theme.

## Key Learnings
> Managing complex control flows with nested if and else statements.
> Using logical operators such as and, or, and not to refine conditions.
> Normalizing user inputs to ensure a smooth user experience.
> Organizing story paths to avoid redundant code blocks.

## How to Run
> 1. Open your terminal in WSL.
> 2. Navigate to the Day 3 folder.
> 3. Run the script: python3 main.py

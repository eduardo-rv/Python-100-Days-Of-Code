# Day 8: Caesar Cipher

## Assignment
> Build a program that can encrypt and decrypt text using the Caesar Cipher method. The script allows users to shift characters by a specific number to hide or reveal secret messages.

## Requirements
> Create a single function that handles both encryption and decryption based on user direction.
> Import and display ASCII art from an external file to give the program a unique visual identity.
> Ensure that numbers, symbols, and spaces remain unchanged when the message is being processed.
> Implement the modulo operator to handle shift numbers that are larger than the total number of letters in the alphabet.

## Key Learnings
> Defining functions that accept multiple parameters and using keyword arguments for better readability.
> Using for loops to iterate through strings and update character positions based on list indices.
> Applying mathematical logic to prevent index errors when a shift exceeds the alphabet length.
> Building an interactive user loop that allows the program to restart without exiting the terminal.

## How to Run
> 1. Open your terminal in WSL.
> 2. Navigate to the Day 8 folder.
> 3. Run the script: python3 main.py

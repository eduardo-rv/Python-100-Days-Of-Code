# Day 5: Password Generator

## Assignment
> Create a program that generates a secure and randomized password based on user preferences. The script should allow the user to decide how many letters, symbols, and numbers they want in their final password.

## Requirements
> The program must utilize for loops to iterate through the specific number of characters requested by the user.
> Use the random module to select individual characters from predefined lists of letters, numbers, and symbols.
> Store the selected characters in a list and then shuffle the list to ensure the password pattern is unpredictable.
> Join the final list of characters into a single string and display it as the generated password.

## Key Learnings
> Using for loops with the range function to repeat actions a specific number of times.
> Understanding the difference between a simple loop and a loop that builds a collection.
> Using random.choice to pick a single item and random.shuffle to reorder a list.
> Converting between lists and strings to manipulate data structure and output.

## How to Run
> 1. Open your terminal in WSL.
> 2. Navigate to the Day 5 folder.
> 3. Run the script: python3 main.py

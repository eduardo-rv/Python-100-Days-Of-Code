# Day 4: Rock Paper Scissors

## Assignment
> Build a digital version of the classic Rock Paper Scissors game where the user competes against a computer making a random selection.

## Requirements
> The program must import the random module to generate a choice for the computer.
> Store the ASCII art representations for Rock Paper and Scissors in a list for easy access via index.
> Ask the user to input a number representing their choice and handle potential invalid entries.
> Implement a logical comparison to determine the winner based on the rules of the game.

## Key Learnings
> Generating random integers and utilizing them to select items from a collection.
> Creating and managing data structures like Python lists.
> Accessing elements within a list using their index and understanding zero-based indexing.
> Using nested conditional statements to compare different outcomes and tie scenarios.

## How to Run
> 1. Open your terminal in WSL.
> 2. Navigate to the Day 4 folder.
> 3. Run the script: python3 main.py

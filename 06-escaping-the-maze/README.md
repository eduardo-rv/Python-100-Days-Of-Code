# Day 6: Escaping the Maze

## Assignment
> Create a set of instructions and logic to help a robot navigate and escape a randomly generated maze. The program focuses on using loops and functions to automate complex movements and pathfinding.

## Requirements
> The program must use a while loop to keep the robot moving until it reaches the specific goal coordinates.
> Define and use custom functions to handle repetitive tasks such as turning right or jumping over hurdles.
> Implement conditional logic to check for walls on the right and in front of the robot at every step.
> Ensure the robot can handle different maze configurations without crashing into walls or getting stuck in loops.

## Key Learnings
> Understanding the fundamental differences between while loops and for loops in Python.
> Defining and calling functions to make code more modular and easier to read.
> Developing algorithmic thinking to solve pathfinding problems in a dynamic environment.
> Managing infinite loops by ensuring the exit condition eventually becomes true.

## How to Run
> 1. Open your terminal in WSL.
> 2. Navigate to the Day 6 folder.
> 3. Run the script: python3 main.py

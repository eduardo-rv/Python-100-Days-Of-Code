# Day 7: Hangman

## Assignment
> Recreate the classic Hangman word guessing game. The program selects a random word from a list and challenges the user to guess it letter by letter before they run out of lives.

## Requirements
> Use the random module to pick a secret word from an external list of words.
> Implement a while loop that continues the game until the user either guesses the full word or loses all six lives.
> Create a display that shows underscores for hidden letters and reveals them as the user makes correct guesses.
> Import ASCII art and the word database from separate Python files to keep the main script organized and modular.

## Key Learnings
> Breaking down complex problems into smaller flowcharts and steps before coding.
> Using the in keyword to check if a specific character exists within a string or list.
> Manipulating lists by replacing specific elements at a given index based on user input.
> Improving code organization by importing variables and constants from external modules.

## How to Run
> 1. Open your terminal in WSL.
> 2. Navigate to the Day 7 folder.
> 3. Run the script: python3 main.py
